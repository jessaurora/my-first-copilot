# 🤖 Meu Primeiro Copilot

Esse repositorio foi criado como parte do curso da DIO sobre GitHub Copilot.
A ideia é aprender a usar os diferentes modos de interação com IA pra melhorar o fluxo de desenvolvimento.

---

## 📁 Estrutura do Projeto

```
my-first-copilot/
│
├── prompts/
│   ├── prompt-ask.md      # Modo para tirar duvidas
│   ├── prompt-edit.md     # Modo para editar codigo
│   ├── prompt-plan.md     # Modo para planejar tarefas
│   ├── prompt-agent.md    # Modo autonomo
│   └── prompt-study.md    # Modo tutor/aprendizado
│
└── readme.md
```

---

## 🧩 Os 5 Modos do Copilot

### ❓ Ask
Use quando quiser **entender** algo sem mexer no codigo.
Perfeito pra tirar duvidas sobre erros, conceitos ou funcionalidades.

### ✏️ Edit
Use quando quiser **modificar** um trecho de codigo existente.
Refatoração, correção de bugs, melhoria de performance...

### 🧭 Plan
Use quando a tarefa for **grande ou complexa**.
O copilot vai planejar tudo antes de sair codando, assim voce valida a abordagem primeiro.

### 🤖 Agent
O modo mais **autonomo** de todos.
Ele navega pelo projeto, cria arquivos, modifica varios pontos e mantem o contexto entre os passos.

### 📚 Study
Use quando quiser **aprender** de verdade, nao so copiar a resposta.
Funciona como um tutor que guia o seu raciocinio.

---

## 🚀 Como usar

1. Copie o conteudo do prompt que quiser usar
2. Cole no chat do GitHub Copilot (ou qualquer outra IA)
3. Comece a interagir!
